### 文档
    https://github.com/vuejs/vue-cli/tree/v2

### 卸载其他脚手架

npm uninstall @vue/cli -g  

### npm清除缓存

    npm cache clean -f

### 安装脚手架
    npm install vue-cli -g

### 查看vue脚手架的版本
    vue -V

### 查看脚手架可以使用的模板
    vue list

### 使用脚手架创建项目
    vue init <template-name> <project-name>
    vue init webpack demo
